## Quick Start

1. **Install Continue** in VS Code:
   ```bash
   code --install-extension continue.continue
   ```

2. **Copy config** to project root:
   ```bash
   cp platforms/continue/.continuerc.json /path/to/your/project/
   ```

3. **Start using** (AI auto-generates files):
   ```
   init
   ```
